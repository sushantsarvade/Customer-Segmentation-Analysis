# Customer Segmentation Analysis Project

## Project Overview

- This project focuses on analyzing customer demographic and purchasing behavior data using Exploratory Data Analysis (EDA) techniques. The objective is to identify patterns in customer income, age, and spending behavior to support data-driven marketing strategies.

## Dataset Information

- CustomerID – Unique identifier
- Gender – Gender of the customer
- Age – Customer age
- Annual_Income_(k$) – Annual income in thousand dollars
- Spending_Score – Score assigned based on purchasing behavior

## Tools used in the project

- Python

- Pandas

- NumPy

- Matplotlib

- Seaborn

- Jupyter Notebook

## Problem Statement

- Identify high value customers.

- Find customers who spend less.

- Target specific groups with offers.


## Project Workflow

### Data Loading

- Imported dataset using Pandas

- Reviewed structure and data types

### Data Cleaning

- Checked for missing values

- Removed duplicate records

- Renamed columns for better readability

### Exploratory Data Analysis (EDA)

- Age distribution

- Gender distribution

- Income distribution

- Spending score distribution

- Income vs Spending behavior

- Age vs Spending trends

- Correlation analysis

## Key Insights

- Majority of customers fall between 20–40 years of age

- Female customers form a slightly larger portion of the customer base

- Spending behavior is not directly proportional to income

- Younger customers tend to have higher spending scores

- A premium segment exists with high income and high spending potential

- High-income but low-spending customers represent upselling opportunities


## Conclusions

- The exploratory data analysis revealed meaningful patterns in customer age, income, and spending behavior. The findings show that spending is not solely dependent on income and that younger customers tend to spend more. These insights can help businesses design targeted marketing strategies and improve customer engagement through data-driven decisions.
